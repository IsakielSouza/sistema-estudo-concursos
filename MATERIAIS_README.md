## 🚀 Como Usar

### **1. Visualizar Materiais**
- Acesse `/materials` para ver todos os materiais
- Use os filtros por categoria e tipo de concurso
- Busque por palavras-chave no campo de busca
- Visualize estatísticas de materiais disponíveis

### **2. Adicionar Novos Materiais**
- Acesse `/materials/upload` (requer login)
- Preencha as informações do material
- Cole a URL do PDF no GitHub
- Selecione categoria e tipo de concurso
- Adicione tags e autor (opcional)

### **3. Acessar PDFs**
- Clique em "Visualizar" para abrir o PDF
- Clique em "Baixar" para baixar o arquivo
- URLs diretas para GitHub (100% gratuito)

1. **Faça upload** do PDF para um repositório GitHub
2. **Clique no arquivo** PDF no GitHub
3. **Clique em "Raw"** ou "Download"
4. **Copie a URL** da barra de endereços
5. **Cole no sistema** de materiais
