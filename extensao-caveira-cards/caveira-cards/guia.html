CaveiraCards

Guia de instalação
Siga os passos abaixo para conectar a extensão ao seu Anki e começar a capturar questões automaticamente.
⚠️ O CaveiraCards se comunica com o Anki via AnkiConnect. O Anki precisa estar aberto e com o plugin instalado para a extensão funcionar.
1

Instale o Anki

Baixe e instale o Anki gratuitamente. É o software onde seus cards de estudo ficam armazenados.

↓ Baixar Anki
2

Instale o plugin AnkiConnect

O AnkiConnect permite que a extensão se comunique com o Anki via API local.

  1. Abra o Anki
  2. Vá em Ferramentas → Complementos → Baixar complementos
  3. Cole o código abaixo e clique em OK
  4. Reinicie o Anki

Ver no AnkiWeb
3

Configure o AnkiConnect para todos os sites

Por padrão, o AnkiConnect só aceita conexões do TEC Concursos. Para funcionar também com Gran Questões (e futuros sites), você precisa atualizar o arquivo de configuração.

  1. No Anki, vá em Ferramentas → Complementos
  2. Selecione AnkiConnect e clique em Config
  3. Substitua o conteúdo pelo JSON abaixo e clique em OK
  4. Reinicie o Anki

4

Crie o modelo de card "CaveiraCards"

O CaveiraCards usa um modelo próprio no Anki com 3 campos: Frente, Verso e Extra.

  1. No Anki, vá em Ferramentas → Gerenciar tipos de notas
  2. Clique em Adicionar → selecione Básico
  3. Renomeie para CaveiraCards
  4. Vá em Campos e edite para ter exatamente: Frente, Verso, Extra
  5. Clique em Salvar

⚠️ Os nomes dos campos precisam ser exatamente esses — com letras maiúsculas.

💀

Tudo pronto!

Com o Anki aberto, acesse o TEC Concursos ou Gran Questões, responda uma questão e o botão do CaveiraCards aparecerá automaticamente na tela.
