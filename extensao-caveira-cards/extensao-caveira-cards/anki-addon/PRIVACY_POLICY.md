# Política de Privacidade - CaveiraCards

A presente Política de Privacidade descreve como a extensão **CaveiraCards** lida com as informações dos usuários. Nossa prioridade é a transparência e a segurança dos seus dados de estudo.

## 1. Coleta de Dados
A extensão **CaveiraCards** não coleta, armazena ou transmite informações de identificação pessoal (PII). 

Os únicos dados processados pela extensão são:
- **Conteúdo de Questões:** Texto da pergunta, alternativas e resposta correta das plataformas de concursos suportadas (TEC Concursos, Gran Questões, etc.).
- **Configurações da Extensão:** Nomes de baralhos e preferências de interface, que são salvos localmente no seu navegador.

## 2. Uso dos Dados
Os dados das questões são capturados apenas quando o usuário interage com a extensão (ex: ao clicar no botão de salvar) e são utilizados exclusivamente para a criação de flashcards no aplicativo **Anki**.

## 3. Armazenamento e Compartilhamento
- **Processamento Local:** Todo o processamento de dados ocorre localmente no computador do usuário.
- **Sem Servidores Externos:** Não enviamos seus dados para servidores externos ou bancos de dados em nuvem.
- **Comunicação com o Anki:** A comunicação dos dados é feita via API local (AnkiConnect) diretamente para o software Anki instalado na sua máquina (`http://localhost:8765`).
- **Sem Venda de Dados:** Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins publicitários ou qualquer outro propósito.

## 4. Permissões do Navegador
A extensão solicita as seguintes permissões estritamente para o seu funcionamento:
- `activeTab`: Para ler o conteúdo da questão na página aberta.
- `storage`: Para salvar suas preferências de baralhos do Anki.
- `notifications`: Para avisar quando um card foi criado com sucesso.
- `host_permissions`: Para permitir a comunicação técnica com os sites de questões e com o Anki local.

## 5. Alterações nesta Política
Esta política pode ser atualizada ocasionalmente para refletir mudanças na extensão ou requisitos legais. Recomendamos a revisão periódica desta página.
